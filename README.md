# Chat-Application
A modern real-time chat application project built in Android Studio, code being written in Java. Easy to use and interactive interface, smooth scrolls makes it a package.It also contains features as camera access, you can send multiple things, and has a room for customization in case you want to add more features. This project was an attempt by me to acquire full knowledge about android chat applications. I have used chat stream.io api.
